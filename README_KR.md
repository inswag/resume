## 활용가능 기술

### iOS App
- Swift 를 활용한 어플리케이션 제작
- Cocoapods, Swift Package Manager 의 활용
- FireBase 를 활용한 어플리케이션 제작
- MySQL, CoreData 를 활용한 로컬 DB 구현
- MVC, MVVM 을 활용한 아키텍처 설계
