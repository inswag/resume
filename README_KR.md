## 활용가능 기술
### iOS
- Swift 4 를 활용한 Application 제작
- Cocoapods, Carthage 를 사용하여 Open Source Library 적용
- FireBase 를 통한 Real-time DB의 동작 이해
- Design Pattern 을 활용하여 앱의 기본적 설계 구현
 
### 도구
